# DinoCloud Server Core

DinoCloud — это система облачного хранения с открытым исходным кодом, ориентированная на защиту приватности и совместную работу. Коллекции файлов называются библиотеками, и каждую библиотеку можно синхронизировать отдельно. Библиотека также может быть зашифрована паролем, выбранным пользователем. DinoCloud позволяет пользователям создавать группы и легко делиться файлами внутри групп.

Это основной компонент сервера DinoCloud. Он предоставляет RPC-интерфейс для веб-фронтенда (Seahub) для доступа к файлам и HTTP API для настольных клиентов для синхронизации файлов.
